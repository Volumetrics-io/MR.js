mrjs volumetrics Camera The camera can be setup with a few different modifications based on user's purpose. On `mr-app` a camera is setup at the default starting position. A user can note the startingPositon as well instead. If using orbital controls, a user can also denote the target position of those controls. This page is its own example and is currently running with the following setup: `orbital="mode:true; targetPos:0 1 0;" camera="mode: perspective; startPos:4 3 1;"`
