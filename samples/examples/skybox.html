mrjs volumetrics Skybox Skyboxes can enhance your spatial app by making it feel more like a unique virtual environment. They can be used to set the scene, bridge virtual and non-virtual situations, and set an ambiance for an area. NOTE: This effect is better viewed directly in an MR headset. Examples Night Sky Starmap Cycle mr-skybox Skybox setup is achieved using the skybox component, `mr-skybox`, it has one attribute: src This src tag is a singular skybox image file.
