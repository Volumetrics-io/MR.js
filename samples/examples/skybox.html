mrjs volumetrics Skybox Skyboxes can enhance your spatial app by making it feel more like a unique virtual environment. They can be used to set the scene, bridge virtual and non-virtual situations, and set an ambiance for an area. Examples Night Sky - a singular image Daytime - multiple images mr-skybox Skybox setup is achieved using the skybox component, `mr-skybox`, it has one attribute: src This src tag can be to singular skybox image file or to a folder with a list of textures image files within it. Internally the system handles how they are loaded so the scene looks right.
